# Retail Sales Dashboard – Power BI

## Overview
This project presents a Retail Sales Dashboard built using Power BI. The dataset was cleaned and prepared before visualization.

## Key Insights
- Sales by Region
- Profit by Product
- Average Profit Margin (KPI)
- Interactive slicer for dynamic filtering

## Tools Used
- Power BI Desktop
- CSV dataset
- Basic data modeling and DAX

## Files Included
- RETAIL DATA.pbix – Power BI dashboard file
- cleaned_retail_data.csv – Dataset used
